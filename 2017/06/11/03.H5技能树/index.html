  • 学习笔记

系统地整理一下H5学习线路图、技能树，先把大纲放出来，学习笔记后续会陆续补全，本文目的：

  • 巩固知识
  • 方便查阅
  • 纯属喜欢把事物整理得井井有条

未来会继续完善技能树，毕竟 学无止境。

首先，工欲善其事必先利其器。所以需要先选择一个称手的武器，也就是编辑器，那么众多的编辑器该如何选择呢？这里简单列举一下开发人员使用率较高的几款编辑器：

  • sublime Text : 收费软件，70刀，但是可以无限期的试用。小巧，轻量级，启动快，插件众多，高度可配置。
  • Hbuilder : 国产优秀IDE，功能全，拥有很丰富的配置选项。
  • VS Code : 来自微软，自带debug模块和git模块，拥有丰富的插件，拥有Nodejs调试面板。
  • ATOM ：来自github的编辑器，布局强大，庞大的插件库。
  • WebStorm : 收费软件，重量级，启动慢，代码提示和快捷键完备，省事。

注：排名不分先后，反正我用的是 WebStorm 和 VS Code ，别问我为什么，因为懒。
