jQuery中的is(':visible')与鼠标事件

  • jQuery

1.jQuery中的is(‘:visible’)是个啥东东，干啥的？

:visible 选择器选取每个当前是可见的元素。
除以下几种情况之外的元素即是可见元素：

  • 设置为 display:none
  • type=”hidden” 的表单元素
  • Width 和 height 设置为 0
  • 隐藏的父元素（同时隐藏所有子元素）

来个demo比较好理解：

效果图如下：

  • 例：card.is(':visible')是指可见的card元素.
  • 如果card元素可见，则调用card.slideUp();使之隐藏.
  • else则是card元素不可见的情况下，调用card.slideDown()；使之显示.
  • 类似的写法还有card.is(‘:first’);card.is(‘:last’);之类的,类似于css的伪类a:hover

2.mouseover、mouseout和mouseenter、mouseleave之间的区别？

mouseover、mouseenter：事件在鼠标指针移动到指定元素上时触发。
mouseout、mouseleave：在鼠标指针移出元素上时触发事件。
区别：

  • onmouseover和mouseout事件在鼠标指针进入元素时触发,但是在其子元素上也会触发(例:p span)，也就是会触发常说的冒泡事件。
  • mouseenter和mouseleave事件则不支持冒泡，所以不会触发冒泡。

终止事件冒泡有个简单的方法：在相应的处理函数内，加入 event.stopPropagation() ,终止事件的广播分发，这样事件停留在本节点，不会再往外传播了。
